# TP19 - Architecture Microservices avec Spring Cloud

Ce projet implémente une architecture microservices basée sur Spring Boot et Spring Cloud pour la gestion de clients et de voitures.

##  Description

Ce projet est une application de démonstration d'architecture microservices qui gère :
- **Clients** : Service de gestion des clients
- **Voitures** : Service de gestion des voitures associées aux clients
- **Service Discovery** : Utilisation d'Eureka pour la découverte de services
- **API Gateway** : Point d'entrée unique pour toutes les requêtes
